# alpaca-map 🦙 🗺

Visualising alpacas on a map

## Install app 🐣

```
npm install
```

## Start app 🚀

```
npm start
```

## Use app 🍎

- Navigate to [http://localhost:3030/](http://localhost:3030/) - Check console.log for messages

- http://localhost:3030/api - Shows message
